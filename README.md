# Water Quality Risk Classification

## 📊 Project Overview
This project analyzes water quality using machine learning to classify water as Safe or Unsafe.

## 🔍 Models Used
- Logistic Regression
- Decision Tree

## 📈 Techniques
- Data Cleaning
- EDA Visualization
- Classification
- Clustering (KMeans, Agglomerative)
- PCA

## 📊 Results
- Decision Tree achieved highest accuracy
- Clustering identified pollution zones

## 🚀 Future Scope
- Real-time water monitoring
- Integration with IoT sensors
